
# 将 PWIntegration 和 TranqolIntegration 内联到 TQLSlib.java

## 分析结果

### PWIntegration.java 的功能
- 向 ClassMap 注册各种类（带 tqls- 前缀）
- 包括：PowerTower、ExtentionStatus、各种方块类、TQDrawf、TQUtls、TranqolIntegration

### TranqolIntegration.java 的功能
- load()：初始化 TQDrawf 和 TQUtls
- init()：空的事件注册（无实际功能）

## 执行步骤

1. 修改 TQLSlib.java
   - 在 loadContent() 中直接添加 ClassMap 注册逻辑
   - 直接调用 TQDrawf.init() 和 TQUtls.init()
   - 移除对 PWIntegration 和 TranqolIntegration 的调用
   - 添加必要的 import

2. 删除 PWIntegration.java

3. 删除 TranqolIntegration.java

4. 编译验证
